## ðŸ“Œ Project Overview

This project is a robust **End-to-End (E2E) Automation Testing Framework** designed for the Flipkart e-commerce application. It simulates complex user journeys including product searching, filtering, window switching, and cart management.

The framework is built using the **Hybrid Framework** approach, seamlessly combining **Behavior Driven Development (BDD)** using Cucumber with the **Page Object Model (POM)** design pattern. This architecture ensures code reusability, maintainability, and scalability.

## ðŸ§ª Test Scenarios Covered

The framework currently automates the following critical business flows:

1.  **Product Search & Sort**
    * Search for "iPhone 15".
    * Sort results by "Price -- Low to High".
    * Handle dynamic page loads.
    * Select the first product and switch focus to the new browser tab.

2.  **Category Navigation & Filtering**
    * Hover over the "Fashion" category menu.
    * Navigate to "Men's Casual Shoes".
    * Apply brand filters (e.g., "Nike") using dynamic locators.
    * Verify filter application via URL and UI validation.

3.  **Add to Cart Journey**
    * Search for "Mobile 5g".
    * Select a product.
    * Switch tabs (Parent/Child window handling).
    * Add the item to the shopping cart.
    * Verify product presence in the cart.

## ðŸ§© Key Technical Implementations

* **JavascriptExecutor:** Implemented to handle "Stale Element References" and click intercepted elements (e.g., hidden menus or overlapping popups).
* **Explicit Waits:** Used `WebDriverWait` to handle dynamic AJAX loading on Flipkart.
* **Window Handling:** Robust utility to switch focus between Parent and Child tabs during product selection.
* **Generic Locators:** Usage of `contains()`, `sibling`, and attribute-based XPaths to handle dynamic class names generated by modern frontend frameworks (React/Angular).

## ðŸš€ Getting Started

### Prerequisites

Ensure you have the following installed on your machine:

* **Java JDK 17** or higher (Set `JAVA_HOME` environment variable).
* **Maven** (Set `MAVEN_HOME` environment variable).
* **Eclipse IDE** (with "Cucumber for Java" and "TestNG" plugins installed) or IntelliJ IDEA.

### Installation Steps

1. **Clone the repository:**
```bash
git clone [https://github.com/yourusername/flipkart-automation-framework.git](https://github.com/yourusername/flipkart-automation-framework.git)

```


2. **Import into IDE:**
* Open Eclipse.
* Go to `File > Import > Maven > Existing Maven Projects`.
* Select the project folder and click **Finish**.


3. **Update Dependencies:**
* Right-click the project > `Maven` > `Update Project`.



### Configuration

Configure the browser and target URL in `src/test/resources/Config/config.properties`:

```properties
url=[https://www.flipkart.com](https://www.flipkart.com)
browser=chrome

```

## â–¶ï¸ How to Run Tests

### Option 1: Via Eclipse (TestRunner)

1. Navigate to `src/test/java/com/testrunner/TestRunner.java`.
2. Right-click the file.
3. Select **Run As > TestNG Test**.

### Option 2: Via Command Line (Maven)

Open a terminal in the project root directory and run:

```bash
mvn test

```
